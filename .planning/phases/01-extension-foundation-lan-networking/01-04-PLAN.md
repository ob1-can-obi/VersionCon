      ```typescript
      interface WizardState {
        step: 1 | 2 | 3 | 4; // 4 = "share with team" completion screen
        sessionName: string;
        port: number;
        networkInterface: string;
        availableInterfaces: Array<{ name: string; address: string }>;
        maxPayloadMB: number; // UI shows MB, converts to bytes for SessionConfig
        inviteCode: string;
        hostIp: string;
        isSessionActive: boolean;
        error: string | null;
      }
      ```
    - Export class `WizardPanel`:
      - Static `currentPanel: WizardPanel | undefined`
      - Private `panel: vscode.WebviewPanel`
      - Private `state: WizardState`
      - Private `sessionHost: SessionHost | null = null`
      - Private `discoveryManager: DiscoveryManager | null = null`
      - Private `context: vscode.ExtensionContext`
      - Private `disposables: vscode.Disposable[] = []`

      - Static `createOrShow(context: vscode.ExtensionContext): void`:
        - If currentPanel exists, reveal it and return
        - Auto-detect values:
          - `const interfaces = getAllIPv4Addresses()`
          - `const primaryIp = getLocalIPv4()`
          - `const freePort = await findFreePort()`
          - `const inviteCode = generateInviteCode()`
        - Create initial state with auto-detected values
        - Create new `vscode.window.createWebviewPanel('versioncon.wizard', 'VersionCon: Host Session', vscode.ViewColumn.One, { enableScripts: true, retainContextWhenHidden: false, localResourceRoots: [vscode.Uri.joinPath(context.extensionUri, 'src', 'ui', 'webview', 'wizard')] })`
        - NOTE: `retainContextWhenHidden: false` -- we rely on stateless pattern, not memory retention
        - Set panel HTML via `getWebviewContent()`
        - Register message listener

      - Private `getWebviewContent(): string`:
        - Generate nonce via `crypto.randomBytes(16).toString('base64')`
        - Construct Content Security Policy: `default-src 'none'; style-src ${webview.cspSource} 'nonce-${nonce}'; script-src 'nonce-${nonce}';`
        - Get webview URIs for wizard.css and wizard.js using `panel.webview.asWebviewUri()`
        - Return HTML string with:
          - CSP meta tag
          - Link to wizard.css with nonce
          - Container div `<div id="wizard-root"></div>`
          - Script tag for wizard.js with nonce
        - NOTE: Do NOT inline any CSS or JS -- all must be in external files loaded via nonce

      - Private `handleMessage(msg: any)`:
        - `webview-ready`: send full state via `postMessage({ type: 'state-update', payload: this.state })`
        - `wizard-next`: update state from msg.payload (step-specific validation), advance step, send state-update
        - `wizard-back`: decrement step (min 1), send state-update
        - `wizard-complete`: finalize session creation:
          1. Build `SessionConfig` from state (convert maxPayloadMB to bytes: `maxPayloadMB * 1024 * 1024`)
          2. Create `SessionHost` with config and host display name
          3. `const actualPort = await sessionHost.start()`
          4. Update state: `port = actualPort`, `step = 4`, `isSessionActive = true`
          5. Store invite code: `secretStore.storeInviteCode(state.sessionName, state.inviteCode)`
          6. Publish mDNS: `discoveryManager.publishSession(state.sessionName, actualPort)`
          7. Send state-update to webview (now shows "share with team" screen)
          8. If error: set state.error, send state-update, do NOT advance step
        - `copy-to-clipboard`: `vscode.env.clipboard.writeText(msg.payload.text)`
        - `wizard-override-port`: update state.port, send state-update
        - `wizard-override-interface`: update state.networkInterface + hostIp, send state-update

      - Private `dispose()`:
        - WizardPanel.currentPanel = undefined
        - NOTE: Do NOT stop SessionHost here -- the session should keep running after wizard panel is closed
        - Dispose panel and disposables

      - Getter `getSessionHost(): SessionHost | null` (for Plan 06 to wire into extension state)
      - Getter `getDiscoveryManager(): DiscoveryManager | null`
