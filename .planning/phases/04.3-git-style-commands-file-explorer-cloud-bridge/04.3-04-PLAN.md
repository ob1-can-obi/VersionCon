<objective>
Build the cloud-bridge surface (SPEC SC-6 + SC-7):
- `versioncon.exportToGitRemote` — host runs `git init` + commit + push inside `.versioncon/branches/{branch}/` to ship the project to a real Git remote (GitHub, GitLab, local bare repo, anywhere git can push).
- `versioncon.importFromGitRemote` — host clones a Git remote into a fresh `.versioncon/branches/{newBranchName}/` and registers it via BranchManager. Use case: starting v2 of a project from a previously-exported v1 remote.

Both commands are admin-gated (host-only via `activeHost !== null` + `permissions.canCreateBranch` proxy, matching the existing host-bypass invariant). All git invocations go through a dedicated `GitBridge` service that uses `child_process.spawn(argv, { shell: false })` — never shell concatenation. URL and branch name inputs are validated against an allowlist regex before they ever reach spawn.

Purpose: Phase 4 UAT user feedback was explicit — "Alice who has the permissions, will take the final branch after testing - will push it to the cloud or wherever they want to use the code at." This plan delivers the LAN → cloud handoff that closes the lifecycle loop.

Output: New GitBridge service + two host-only commands + Output channel for visibility + regression coverage including a real round-trip against a local bare repo.
</objective>

<allowlist_regex>
URL allowlist (T-04.3-03 mitigation):
- `^https://[A-Za-z0-9._-]+(?::[0-9]+)?/[A-Za-z0-9._/-]+(?:\.git)?$` — https URLs (github.com, gitlab.com, custom hosts)
- `^git@[A-Za-z0-9._-]+:[A-Za-z0-9._/-]+(?:\.git)?$` — ssh URLs (git@github.com:user/repo.git)
- `^file://(?:/[A-Za-z0-9._/-]+)+(?:\.git)?$` — local bare repos (for tests + advanced users)
- ssh:// optional — defer to v2; https + ssh-shorthand + file:// is enough for v1

Branch name allowlist: `^[A-Za-z0-9._/-]{1,128}$` — alphanumeric + dot + slash + dash + underscore, no spaces, no shell metachars, max 128 chars

Reject ANY input that fails the regex with an error message before spawn is ever invoked.

1. Create `src/services/GitBridge.ts`:
   ```typescript
   import { spawn } from 'child_process';
   import * as fs from 'fs/promises';
   import * as path from 'path';

   const URL_ALLOWLIST = [
     /^https:\/\/[A-Za-z0-9._-]+(?::[0-9]+)?\/[A-Za-z0-9._/-]+(?:\.git)?$/,
     /^git@[A-Za-z0-9._-]+:[A-Za-z0-9._/-]+(?:\.git)?$/,
     /^file:\/\/(?:\/[A-Za-z0-9._/-]+)+(?:\.git)?$/,
   ];
   const BRANCH_NAME_ALLOWLIST = /^[A-Za-z0-9._/-]{1,128}$/;

   export interface GitResult {
     stdout: string;
     stderr: string;
     code: number;
   }

   export interface ExportOpts {
     branchDir: string;        // absolute path to .versioncon/branches/{branch}/
     remoteUrl: string;        // validated by caller via validateUrl
     branchOnRemote: string;   // validated by caller via validateBranchName
     commitMessage: string;    // arbitrary string, passed via spawn argv (no shell)
     onOutput: (line: string) => void; // streaming output for Output channel
   }

   export interface ImportOpts {
     remoteUrl: string;
     branchOnRemote: string;
     destDir: string;          // absolute path where the clone contents land
     onOutput: (line: string) => void;
   }

   export class GitBridge {
     validateUrl(url: string): boolean {
       if (typeof url !== 'string' || url.length === 0 || url.length > 2048) return false;
       return URL_ALLOWLIST.some(re => re.test(url));
     }

     validateBranchName(name: string): boolean {
       if (typeof name !== 'string') return false;
       return BRANCH_NAME_ALLOWLIST.test(name);
     }

     async runGit(args: string[], cwd: string): Promise<GitResult> {
       return new Promise((resolve, reject) => {
         const child = spawn('git', args, { cwd, shell: false });  // T-04.3-03: explicit shell:false
         let stdout = '';
         let stderr = '';
         child.stdout?.on('data', (b: Buffer) => { stdout += b.toString(); });
         child.stderr?.on('data', (b: Buffer) => { stderr += b.toString(); });
         child.on('error', (err) => reject(err));   // git not on PATH, etc.
         child.on('close', (code) => resolve({ stdout, stderr, code: code ?? -1 }));
       });
     }

     async exportToRemote(opts: ExportOpts): Promise<void> {
       // Pre-validate (defense-in-depth; caller already validated)
       if (!this.validateUrl(opts.remoteUrl)) throw new Error('Invalid remote URL');
       if (!this.validateBranchName(opts.branchOnRemote)) throw new Error('Invalid branch name');
       const cwd = opts.branchDir;
       const run = async (args: string[]): Promise<GitResult> => {
         opts.onOutput(`$ git ${args.join(' ')}`);
         const r = await this.runGit(args, cwd);
         if (r.stdout) opts.onOutput(r.stdout);
         if (r.stderr) opts.onOutput(r.stderr);
         opts.onOutput(`[exit ${r.code}]`);
         return r;
       };
       // Idempotent init — `git init` on an existing repo is safe.
       let r = await run(['init']);
       if (r.code !== 0) throw new Error(`git init failed: ${r.stderr.trim()}`);
       r = await run(['add', '.']);
       if (r.code !== 0) throw new Error(`git add failed: ${r.stderr.trim()}`);
       // Use -m via argv — commitMessage NEVER hits a shell.
       r = await run(['commit', '-m', opts.commitMessage]);
       // commit may exit 1 if there's nothing to commit (empty branch dir);
       // treat that as soft success and continue to push.
       if (r.code !== 0 && !r.stdout.includes('nothing to commit')) {
         // Not all git versions put 'nothing to commit' in stdout; also check stderr.
         if (!r.stderr.includes('nothing to commit')) {
           throw new Error(`git commit failed: ${r.stderr.trim() || r.stdout.trim()}`);
         }
       }
       // remote add origin — if origin already exists, set-url instead.
       r = await run(['remote', 'add', 'origin', opts.remoteUrl]);
       if (r.code !== 0) {
         r = await run(['remote', 'set-url', 'origin', opts.remoteUrl]);
         if (r.code !== 0) throw new Error(`git remote add/set-url failed: ${r.stderr.trim()}`);
       }
       r = await run(['push', '-u', 'origin', opts.branchOnRemote]);
       if (r.code !== 0) throw new Error(`git push failed: ${r.stderr.trim()}`);
     }

     async importFromRemote(opts: ImportOpts): Promise<void> {
       if (!this.validateUrl(opts.remoteUrl)) throw new Error('Invalid remote URL');
       if (!this.validateBranchName(opts.branchOnRemote)) throw new Error('Invalid branch name');
       // Ensure destDir does not pre-exist (we expect BranchManager to NOT
       // have created it yet — that's the caller's contract).
       try {
         await fs.access(opts.destDir);
         throw new Error(`Destination already exists: ${opts.destDir}`);
       } catch (err) {
         if ((err as NodeJS.ErrnoException).code !== 'ENOENT') throw err;
       }
       await fs.mkdir(path.dirname(opts.destDir), { recursive: true });
       const args = ['clone', '--branch', opts.branchOnRemote, '--single-branch', opts.remoteUrl, opts.destDir];
       opts.onOutput(`$ git ${args.join(' ')}`);
       const r = await this.runGit(args, path.dirname(opts.destDir));
       if (r.stdout) opts.onOutput(r.stdout);
       if (r.stderr) opts.onOutput(r.stderr);
       opts.onOutput(`[exit ${r.code}]`);
       if (r.code !== 0) throw new Error(`git clone failed: ${r.stderr.trim()}`);
       // Strip .git/ so the imported tree is a plain VersionCon branch dir.
       await fs.rm(path.join(opts.destDir, '.git'), { recursive: true, force: true });
     }
   }
   ```

    * Phase 4.3 SC-7: register a branch whose contents were materialized by an
    * external process (e.g. GitBridge.importFromRemote cloning into the dest
    * dir). Does NOT create or copy the dir — caller has already populated it.
    * Adds the BranchInfo to metadata and persists. Throws if a branch with
    * the same name already exists OR if the dir does not exist on disk.
    */
   async registerExternalBranch(
     name: string,
     creatorId: string,
   ): Promise<BranchInfo> {
     if (!/^[a-zA-Z0-9._/-]+$/.test(name)) {
       throw new Error('Branch name must contain only alphanumeric characters, dots, hyphens, slashes, and underscores');
     }
     if (this.metadata.some(b => b.name === name)) {
       throw new Error(`Branch "${name}" already exists`);
     }
     const dir = path.join(this.branchesDir, name);
     if (!await this.dirExists(dir)) {
       throw new Error(`Branch dir not found on disk: ${dir} — caller must populate before registering`);
     }
     const info: BranchInfo = {
       name,
       createdBy: creatorId,
       createdAt: Date.now(),
       locked: false,
     };
     this.metadata.push(info);
     await this.saveMetadata();
     return info;
   }
